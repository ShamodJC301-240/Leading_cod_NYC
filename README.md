# NYC Leading Causes of Death – Data Pipeline & Analysis

## 📊 Project Overview

This project pulls public health data from the NYC Open Data API and performs data cleaning and exploratory analysis on leading causes of death in New York City.

The goal is to practice real-world data workflows including:
- API data ingestion
- Pagination handling
- Data cleaning and preprocessing
- Structured project organization
- Exploratory data analysis (in progress)

> ⚠️ This project is still in progress and will be completed with full analysis and visualization.

---

## 🧠 Dataset

- Source: NYC Open Data
- Dataset: Leading Causes of Death
- API Endpoint:  
  `https://data.cityofnewyork.us/resource/jb7j-dtam.json`
